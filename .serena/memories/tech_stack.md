# Tech Stack

## Core Framework
- **Next.js 16.1.1**: React framework with App Router
- **React 19.2.3**: UI library
- **TypeScript 5**: Type-safe JavaScript

## Styling
- **Sass (SCSS)**: CSS preprocessor
- **CSS Modules**: Component-scoped styles (.module.scss)
- **Material UI (@mui/material 7.3.6)**: Component library
- **Emotion**: CSS-in-JS for Material UI (@emotion/react, @emotion/styled)
- **Google Fonts**: Geist Sans and Geist Mono

## Authentication
- **Auth.js (NextAuth v5.0.0-beta.30)**: Authentication framework
- **Google OAuth**: Single Sign-On provider

## Development Tools
- **ESLint 9**: Linting with Next.js config
- **TypeScript**: Strict mode enabled
- **Next.js Dev Server**: Hot module replacement

## Project-Specific Technologies
- **Pyodide**: Python interpreter in WebAssembly for Tic-Tac-Toe demo
- **HTML5 Canvas**: For Asteroids game
- **Socket.io**: Real-time communication for Board Race (separate service)
- **Google Cloud Vision API**: Handwriting recognition for Board Race

## Build & Deployment
- **Vercel**: Production hosting for portfolio
- **Railway.app**: Hosting for Board Race backend service
- **Docker**: Containerization for Board Race

## Node Version
- Node 20 (based on @types/node version)

## Package Manager
- npm (package-lock.json present)
