# Project Overview

## Project Name
My Portfolio Site (Evie Marie Kolb's Portfolio)

## Purpose
Personal portfolio website for Evie Marie Kolb, a Full-Stack Software Engineer. The site showcases:
- Professional experience and skills
- Interactive project demos (Tic-Tac-Toe with Python/WebAssembly, Asteroids game, Board Race multiplayer game)
- Personal journey, interests, and hobbies
- Contact information and professional connections

## Key Features
- **Public Pages**: Home, About, Projects, Contact, Journey, Interests, Profile
- **Private Dashboard**: Protected pages accessible via Google SSO (restricted to owner's Gmail)
- **Interactive Demos**: 
  - Tic-Tac-Toe running Python in browser via Pyodide (WebAssembly)
  - Precourse Asteroids game (HTML5 Canvas)
  - Board Race! Ting Xie! (Multiplayer handwriting recognition with Socket.io and Google Cloud Vision API)
- **Authentication**: Google OAuth integration using Auth.js (NextAuth v5 beta)
- **Modern Design**: SCSS-based styling with parallax effects, responsive layouts, and watercolor wash themes

## Target Audience
- Recruiters and hiring managers
- Technical interviewers
- Professional connections

## Deployment
- Portfolio: Vercel
- Board Race backend: Railway.app (separate service)
- Domain: eviemariekolb.com
